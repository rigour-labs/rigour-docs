# Installation

Get Rigour set up in your project in under a minute.

## Prerequisites

- Node.js 18+
- npm, yarn, or pnpm

## Install Globally

```bash
npm install -g rigour
```

## Install as Dev Dependency

```bash
npm install -D rigour
```

## Use with npx (No Install)

```bash
npx rigour check
```

## Verify Installation

```bash
rigour --version
```

You should see output like:

```
rigour v1.6.0
```
